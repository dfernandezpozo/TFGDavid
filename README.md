### Funcionamiento 
En esta parte del ***tfg*** a parte de haberlo empezado de 0 en ***angular** también he implementado un **backend** para que cúando el usuario se registre en AxonAI
automáticamente se guarde el usuario en un json dentro de la carpeta **backend** con la peculiaridad de que las contraseñas que ingresan los usuarios se quedan encriptadas.
### Pruebas
Para probar el trabajo debes hacer cd de la carpeta de **backend** y ahi ejecutar en la terminal ***npm start*** Para que se ponga en marcha el localhost.
En segundo lugar, con la terminal anterior **abierta** debes abrir una nueva, **cd** de ***ChatBotProyect*** y ahi ejecutamos el **ng serve -o**.
